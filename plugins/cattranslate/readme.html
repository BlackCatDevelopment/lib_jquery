Verwendung

Das Plugin wird vom Backend automatisch eingebunden. Es ist die Schnittstelle zu den BlackCat CMS Sprachdateien und dient der Übersetzung von Sprachstrings in Javascript.

Einbinden im Frontend


Am besten über eine headers.inc.php oder alternativ im Template:
<script src="http://localhost/modules/lib_jquery/plugins/cattranslate/cattranslate.js" type="text/javascript"></script>

Synopsis


cattranslate( string, elem, attributes, module )

string - der zu übersetzende Text
elem (optional) - HTML-Element, dessen Text geändert werden soll
attributes (optional) - Attribute, die im Text ersetzt werden sollen
module (optional) - Name des Moduls, dessen Sprachdatei verwendet werden soll

Beispiel


alert(cattranslate('This should be translated into anything else'));
